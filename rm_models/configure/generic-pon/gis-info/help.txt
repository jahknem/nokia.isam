help configure generic-pon gis-info
OLT GIS INFO CONFIGURATION COMMAND
==================================

Command Description
-------------------

This command configures the OLT GIS info.


User Level
----------

The command can be accessed by operators with equipment privileges, and 
executed by operators with equipment privileges.

Command Syntax
--------------

The command has the following syntax:
> configure generic-pon gis-info [ no longitude | longitude 
<Gpon::OltLongitude> ] [ no latitude | latitude <Gpon::OltLatitude> ] [ no 
elevation | elevation <Gpon::OltElevation> ] [ no horizontal-error | 
horizontal-error <Gpon::OltHrzErr> ] [ no altitude-error | altitude-error 
<Gpon::OltAltErr> ] [ no area-code | area-code <Gpon::OltAreaCode> ] [ no 
timestamp | timestamp <Gpon::OltTimestamp> ] [ no gis-digest | gis-digest 
<Gpon::OltGisDigest> ] [ no gis-check | gis-check <Gpon::OltGisCheck> ]

Command Parameters
------------------

Table "OLT GIS info Configuration Command" Command Parameters

Parameter                  Description

[no] longitude             longitude of GIS info
                           optional parameter with default value: "0.00000"

                           Parameter type: <Gpon::OltLongitude>
                           Format:
                           - longitude of the GIS info
                           - unit: degree
                           - range: [-180.00000...180.00000]

                           Parameter type: <Gpon::OltLatitude>
                           Format:
                           - latitude of the GIS info

                           - range: [-90.00000...90.00000]

                           Parameter type: <Gpon::OltElevation>
                           Format:
                           - latitude of the GIS info

                           - range: [-32768...32767]

[no] horizontal-error      horizontal error of GIS info
                           optional parameter with default value: "0"

                           Parameter type: <Gpon::OltHrzErr>
                           Format:
                           - horizontal error of the GIS info

                           - range: [0...65535]

[no] altitude-error        altitude error of GIS info

                           Parameter type: <Gpon::OltAltErr>
                           Format:

                           - range: [0...65535]

[no] area-code             area code of GIS info

                           Parameter type: <Gpon::OltAreaCode>

charaters

[no] timestamp             timestamp of GIS info

[no] gis-digest            GIS digest of GIS info
