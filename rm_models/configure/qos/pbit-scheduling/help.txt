help configure qos pbit-scheduling
P-BIT BASED SCHEDULING FOR SC FORWARDER CROSS CONNECT/RESIDENTIAL BRIDGE 
CONFIGURATION COMMAND
================================================================================
==============

Command Description
-------------------

This command allows the operator to configure the P-bit based scheduling for 
the S+C VLAN forwarder model for Cross Connect / Residential Bridge.


User Level
----------

The command can be accessed by operators with qos privileges, and executed by 
operators with qos privileges.

Command Syntax
--------------

The command has the following syntax:
> configure qos pbit-scheduling [ no sc-xcon-us | sc-xcon-us <Qos::PbitSched> ] 
[ no sc-xcon-dn | sc-xcon-dn <Qos::PbitSched> ] [ no sc-rb-us | sc-rb-us 
<Qos::PbitSched> ] [ no sc-rb-dn | sc-rb-dn <Qos::PbitSched> ]
