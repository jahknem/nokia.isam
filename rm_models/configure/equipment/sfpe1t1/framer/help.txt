2XE1/DS1 FRAMER CONFIGURATION COMMAND
=====================================

Command Description
-------------------

This command allows the operator to configure Framer parameters on the 2xE1/DS1 
sfp:


User Level
----------

The command can be accessed by operators with equipment privileges, and 
executed by operators with equipment privileges.

Command Syntax
--------------

The command has the following syntax:
> configure equipment sfpe1t1 framer (position) [ no frame-mode | frame-mode 
<Eqpt::Framermode> ] [ no iwf-on-liu | iwf-on-liu <Eqpt::IwfOnLiu> ] [ no 
liu-on-liu | liu-on-liu <Eqpt::LiuOnliu> ] [ no pm-to-iwf | pm-to-iwf 
<Eqpt::PmToiwf> ] [ no ais-to-iwf | ais-to-iwf <Eqpt::AisToiwf> ] [ no 
rai-to-iwf | rai-to-iwf <Eqpt::RaiToiwf> ] [ no rei-to-iwf | rei-to-iwf 
<Eqpt::ReiToiwf> ] [ no pm-to-liu | pm-to-liu <Eqpt::PmToliu> ] [ no ais-to-liu 
| ais-to-liu <Eqpt::AisToliu> ] [ no rai-to-liu | rai-to-liu <Eqpt::RaiToliu> ] 
[ no rei-to-liu | rei-to-liu <Eqpt::ReiToliu> ]

------------------

(position)                 the physical sfp cage position
