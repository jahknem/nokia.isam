STATIC OUTBOUND MPLS LSP CONFIGURATION COMMAND
==============================================

Command Description
-------------------

This command allows the operator to configure a static outbound MPLS LSP, 
include the MPLS label and nexthop of an outgoing MPLS tunnel.


User Level
----------

The command can be accessed by operators with drouter privileges, and executed 
by operators with drouter privileges.

Command Syntax
--------------

The command has the following syntax:
> configure dist-router mpls ( no static-lsp (lsp-name) ) | ( static-lsp 
(lsp-name) interface <Drouter::IfIndex> push <Drouter::OutLabel> [ no 
nexthopaddrtype | nexthopaddrtype <Drouter::IpAddressType> ] nexthop 
<Ip::V4AddressHex> )

Command Parameters
------------------

Table "Static Outbound MPLS LSP Configuration Command" Resource Parameters

Resource Identifier        Description

(lsp-name)                 Administrative name for the LSP

Table "Static Outbound MPLS LSP Configuration Command" Command Parameters

Parameter                  Description

interface                  specify the outgoing interface
                           mandatory parameter

                           Parameter type: <Drouter::IfIndex>
                           Format:
                           ( slot-vlan : <Eqpt::RackId> / <Eqpt::ShelfId> / 
<Eqpt::SlotId> : <Eqpt::UnstackedVlan>
                           | slot-stacked : <Eqpt::RackId> / <Eqpt::ShelfId> / 
<Eqpt::SlotId> : <Eqpt::SVlan> : <Eqpt::CVlan> )
                           Possible values:
                           - slot-vlan             : slot unstacked
                           - slot-stacked          : slot stacked
                           Field type <Eqpt::RackId>
                           - the rack number
                           Field type <Eqpt::ShelfId>
                           - the shelf number
                           Field type <Eqpt::SlotId>
                           - the LT slot number
                           Field type <Eqpt::UnstackedVlan>
                           - unstacked vlan id
                           Field type <Eqpt::SVlan>
                           - service vlan id
                           Field type <Eqpt::CVlan>
                           - customer vlan id

push                       specify the outgoing label
                           mandatory parameter

                           Parameter type: <Drouter::OutLabel>
                           Format:
                           - Out Label
                           - range: [1...1048575]

[no] nexthopaddrtype       specify the type of next hop IP address
                           optional parameter with default value: "ipv4"

                           Parameter type: <Drouter::IpAddressType>
                           Format:
                           ( ipv4
                           | ipv6 )
                           Possible values:
                           - ipv4                  : ipv4
                           - ipv6                  : ipv6

nexthop                    Next hop IP address
                           mandatory parameter

                           Parameter type: <Ip::V4AddressHex>
                           Format:
                           - IPv4-address
