help configure alarm filter
ALARM FILTER CONFIGURATION COMMAND
==================================

Command Description
-------------------

This command allows the operator to configure Alarm filter parameters.


User Level
----------

The command can be accessed by operators with alarm privileges, and executed by 
operators with alarm privileges.

Command Syntax
--------------

The command has the following syntax:
> configure alarm filter (fltr-type) filterid <Alarm::fltrId> [ alarmid 
<Alarm::alarmIndexExt> ] [ no status | status <Alarm::fltrStatus> ] [ no 
threshold | threshold <Alarm::fltrThreshold> ] [ no window | window 
<Alarm::fltrWindow> ]

Command Parameters
------------------

Table "Alarm Filter Configuration Command" Resource Parameters

Resource Identifier        Description

(fltr-type)                the type of the considered alarm filter
filterid                   unique filter number
